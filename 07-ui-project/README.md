## рџ§© Component Architecture

The UI is divided into **modular components** following React best practices.

### Section1 Components

* **Navbar.jsx** вЂ“ Top navigation section
* **HeroText.jsx** вЂ“ Main headline and description
* **Leftcontent.jsx** вЂ“ Left side content block
* **RightContent.jsx** вЂ“ Container for cards
* **RightCard.jsx** вЂ“ Individual card layout
* **RightCardContent.jsx** вЂ“ Card text and labels
* **Arrow.jsx** вЂ“ Navigation arrow element
* **Page1content.jsx** вЂ“ Main layout wrapper

This structure improves:

* Reusability
* Maintainability
* Scalability
