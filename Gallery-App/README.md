# 📸 Simple Gallery App

A clean and minimal image gallery application that displays photos in a grid layout with pagination support. Built for learning and practicing frontend concepts like component structure, API integration, and UI rendering.

---

## 🚀 Features

* 🖼️ Responsive image grid layout
* 👤 Displays photographer/author names
* 🔄 Pagination (Next / Prev navigation)
* ⚡ Fast loading with optimized UI
* 🎯 Clean and minimal design

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** CSS / Tailwind (if used)
* **API:** Unsplash API (or any image source)
* **HTTP Client:** Axios / Fetch

---

## 📸 How It Works

* The app fetches images from an API.
* Images are displayed in a grid format.
* Each card shows:

  * Image
  * Photographer name
* Pagination allows navigating between pages.

---

## 🎯 Learning Goals

This project helps in understanding:

* React component structure
* API integration
* State management
* Pagination logic
* UI design principles
