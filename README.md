# ⚛️ React Mini Projects Collection

A collection of small and practical React projects built while learning and exploring core concepts of React. Each project focuses on specific features like state management, event handling, and UI design.

---

## 🚀 Projects Included

### 🧮 Counter Project

* Increment, decrement, and reset functionality
* Demonstrates use of `useState` hook

---

### 📝 Notes App

* Create, edit, and delete notes
* Uses Local Storage for data persistence
* Focus on CRUD operations

---

### 🎨 UI Project

* Basic UI components and layouts
* Focus on styling and responsive design

---
