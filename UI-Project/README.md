# React UI Project вҖ“ Customer Segmentation Interface

A modern responsive UI built using **React + Vite + TailwindCSS**.
This project demonstrates a **component-based architecture** where a large UI section is divided into reusable React components.

The interface showcases a **prospective customer segmentation dashboard** with card-based layouts and modular sections.

---

## рҹҡҖ Tech Stack

* React
* Vite
* TailwindCSS

## рҹ§© Component Architecture

The UI is divided into **modular components** following React best practices.

### Section1 Components

* **Navbar.jsx** вҖ“ Top navigation section
* **HeroText.jsx** вҖ“ Main headline and description
* **Leftcontent.jsx** вҖ“ Left side content block
* **RightContent.jsx** вҖ“ Container for cards
* **RightCard.jsx** вҖ“ Individual card layout
* **RightCardContent.jsx** вҖ“ Card text and labels
* **Arrow.jsx** вҖ“ Navigation arrow element
* **Page1content.jsx** вҖ“ Main layout wrapper

This structure improves:

* Reusability
* Maintainability
* Scalability

## вҡҷпёҸ Installation & Setup
### Clone the repository

```bash
git clone https://github.com/Shubh7amydv/React_Project.git
```

Navigate to the UI project

```bash
cd React_Project/07-ui-project
```

Install dependencies


